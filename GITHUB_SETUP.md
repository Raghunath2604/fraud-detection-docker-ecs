## 📍 Quick Reference

### Clone & Run Locally
```bash
git clone https://github.com/Raghunath2604/fraud-detection-docker-ecs.git
cd fraud-detection-docker-ecs
pip install -r requirements.txt
python train.py
uvicorn app.main:app --reload
```

### Run with Docker
```bash
docker build -t fraud-api .
docker run -p 8000:8000 fraud-api
```
